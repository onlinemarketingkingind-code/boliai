# 🎙️ BoliAI - Speak Freely. Learn Naturally.

## 🌟 Vision

**BoliAI** (formerly PolyLingua AI) is the universal bridge connecting the world to India's linguistic diversity. We're building the smartest AI-powered platform that enables anyone - from anywhere - to learn, speak, and understand any Indian language with cultural context.

### The Problem We Solve

#### 🌍 **Global Challenge**
- International travelers struggle to communicate in regional Indian languages
- Businesses expanding to India face language barriers
- Diaspora communities lose connection with their heritage
- Cultural nuances are lost in generic translation apps

#### 🇮🇳 **Domestic Challenge (45M+ Inter-State Migrants)**
- Tamil worker in Mumbai can't communicate with local shopkeepers
- Bengali student in Bangalore struggles with daily conversations
- UP migrant in Chennai faces workplace communication barriers
- Inter-state families struggle to bridge language gaps

### Our Solution

**Input:** ANY language in the world (English, German, Japanese, Spanish, etc.)  
**Output:** ALL 28+ Indian languages (Hindi, Marathi, Tamil, Bengali, etc.)  
**Context:** Travel, Business, Culture, Education, Daily Life, Emergency

---

## 🎯 Core Features

### 1. 🗺️ **Travel Mode**
Learn phrases for airports, hotels, restaurants, markets, temples, and emergencies with location-specific variations.

### 2. 💼 **Job/Professional Mode**
Master workplace communication with industry-specific vocabulary and formal business etiquette.

### 3. 🎭 **Culture Mode**
Understand festivals, weddings, traditions, and regional customs with respectful, context-aware language.

### 4. 😊 **Casual/Daily Life Mode**
Connect with locals using informal phrases, slang, and everyday conversations.

### 5. 🌉 **Inter-State Bridge Mode** (Bharat Mode)
Help Indians communicate across state boundaries - from survival phrases to cultural integration.

### 6. 🎓 **Grammar & Deep Learning Mode**
Master scripts (Devanagari, Tamil, Bengali), sentence structure, and pronunciation.

## 💡 Getting Started

### Prerequisites
```bash
Node.js 18+
npm or yarn
Supabase account
OpenAI API key (for GPT-4)
Google Cloud account (for TTS & Translation)
```

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/boliai.git
cd boliai

# Install frontend dependencies
cd frontend
npm install

# Install backend dependencies
cd ../backend
npm install

# Set up environment variables
cp .env.example .env
# Add your API keys to .env

# Run development servers
npm run dev
```
